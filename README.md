# 📝 Word Analytics

A smart text analysis tool that counts words, characters, lines, and shows character limits for Twitter and Facebook posts.

---

## ✨ Features

- 📄 **Word count** – Number of words in your text
- 🔤 **Character count** – Total characters (with and without spaces)
- 📏 **Line count** – Number of lines
- 🐦 **Twitter limit** – Remaining characters for a tweet (280)
- 📘 **Facebook limit** – Remaining characters for a Facebook post (2200)

---

## 🎯 How to Use

1. Type or paste your text into the text area
2. All statistics update automatically in real-time
3. Check remaining characters for Twitter and Facebook

---

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (ES6+)
